What is a function

A function is a block of code that perfoms a specific task and functions allow you to organise your code into logical blocks.

for example: function greet(name) { console.log(`Hello, ${name}!`); }

Function call.

A function call is when you execute a function.

e.g , greet('Somila'); // Output: Hello, Yonela

A function deifinition is a code block that:

  • Declares a function's name
  • Specifies its perimeters(inputs)
  • Defines the function's behavior
  • And lastly returns a value

By defining a function, you can call it multiple times with different inputs to produce different inputs to produce different outputs, making the code more efficient and flexible .

Variables

A variable is a container that holds a value like -let name, const age. It's a way to store and label a value in a code .

  • - `let y = 20;` declares a variable `y` and assigns it the value `20`.
  • - `const name = 'Somila';` declares a variable `name` and assigns it the value `'Yonela'`.
  • - `var age = 20;` declares a variable `age` and assigns it the value `20`.

Also you can use variables to store user input, calcylate values, store data from an API variables are a fundamental concept in programming.

Data types

Data types are categories that define the type of data a variable can hold. They determine the type of value a variable can store.
